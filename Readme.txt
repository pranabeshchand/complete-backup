Before running the project Make sure that

1. Run npm install in both root and public directory.

2. Make sure that Typescript is installed and mondodb service is running.

3. Manually install jsonparsertool in root node_modules directory from  https://xp-dev.com/git/jsonparsertool

4. To start the server run "npm start" in project root directory.

5. For Client side code run "npm start" in public directory for conversion *.ts files into *.js files.
   This will create dist folder having *.js files respective of *.ts files.

6. In browser visit this url http://localhost:3000.
